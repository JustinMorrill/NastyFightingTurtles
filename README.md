# NastyFightingTurtles

## Prerequisites

- [Node](https://nodejs.dev/download)

## Install Code
```sh
$ cd dir/to/this/repo/.../app
$ npm install
```
## Run Code
```sh
$ cd dir/to/this/repo/.../app
$ node __init__
```

macOS 12+ specific: open `app/__init__.js` in your text editor of choice and change `const port = 5000;` to `const port = 8000;`
